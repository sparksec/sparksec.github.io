Part 2 or what started me on this hacking obsession

Curiosity is key, you have to have a curious mind, how does this work? what happens if I do this? Can I make it do something it is not intended or designed to do? Can I break it? Can I find a flaw, vulnerability?

But with hacking it is so much more than curiosity it is a huge field with so much to learn, Movies make it look so easy and sexy (well it it is sexy) but it takes a shit ton of hard work to make it happen especially if you are starting from zero.

Luckily I have been in the IT field for more than few years so pretty comfortable with networking windows and Linux command line, bash and python scripting and other stuff I won’t bore you with, so I am probably starting at what I thought was 50 but is probably more like 30 at most. So I reckon it is still going to be a tough road for me.

So rounding up in my opinion to be a good hacker you need to be curious and a willingness to learn and work hard. any other computer related experience is a bonus.

In part 3 I will explaining what I am doing to learn and the resources I am using so see you then……
